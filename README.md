# Modify Request Headers

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

It is a browser extension that adds, overwrites, or removes request headers for URLs you specify.

[![Available in the Chrome Web Store](./images/iNEddTyWiMfLSwFD6qGq.png)](#)

## How to use

1. Click on the icon for this browser extension to open the popup.
2. In the form, choose how to match target requests (exact URL, URL prefix, or regular expression), then enter the URL/pattern.
3. Enter the Header name, Operation (set / append / remove), and Value, then click the save button.
4. From then on, requests matching the pattern will have the header modified automatically.

Saved rules are grouped by their matching pattern and shown in the rule list, where you can toggle a rule active/inactive, edit it, or delete it.
